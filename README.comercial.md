# PayDo - README Comercial

## Proposta de valor

A PayDo oferece pagamentos transfronteiricos instantaneos e seguros para:
- tributos
- contas concessionarias
- convenios de arrecadacao

Conectamos Brasil e Paraguai com operacao simples, rastreavel e escalavel.

## Problema que resolvemos

Empresas e parceiros que operam entre paises enfrentam:
- processos fragmentados
- baixa previsibilidade operacional
- dificuldade de conciliacao
- comprovacao de pagamento pouco confiavel

## Solucao PayDo

- Orquestracao unificada de pagamentos B2B2C
- Execucao em rails locais (sem friccao para o usuario final)
- Confirmacao rapida de status
- Comprovantes e trilha auditavel
- XRPL no backend para prova de integridade

## Para quem

- fintechs
- integradores
- escritorios contabeis/administrativos
- plataformas de arrecadacao e operacao financeira

## Diferenciais

- Foco em obrigacoes reais (nao remessa generica)
- Modelo partner-first para aceleracao de distribuicao
- Arquitetura preparada para escala por corredor
- Auditabilidade robusta sem complexidade para o cliente final

## Escopo inicial

- Corredor: Brasil <-> Paraguai (bidirecional)
- Segmentos: tributos, concessionarias e convenios de arrecadacao
- Modelo: MVP com evolucao planejada para Mercosul

## Indicadores de sucesso (MVP)

- taxa de sucesso de execucao
- tempo ate confirmacao do pagamento
- tempo de emissao de comprovante
- tempo de conciliacao
- percentual de ancoras XRPL concluidas
